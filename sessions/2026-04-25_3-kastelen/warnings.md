# Warnings — 3 kastelen (April 25)

## Long run executed at threshold intensity — wrong session type

**The data:** 92.2% of 12 km run in Z3 (threshold, 143–162 bpm). Average pace 6:17 /km for 76 minutes.

**Why it matters:** Long runs serve a specific purpose: build aerobic base and distance tolerance at easy pace (Z1–Z2, <143 bpm). When you run a long distance at threshold intensity, you get neither benefit:
- Not building aerobic base (that happens below Z2)
- Not recovery (threshold work requires recovery afterward)
- Accumulating fatigue and central nervous system load

This wasn't a bad run (execution was perfect — zero drift, consistent pacing), but it was the wrong type of run for this session.

**Going forward:** Next 12+ km run should be Z1–Z2 easy pace (6:45–7:30 /km, <143 bpm HR). You'll feel slower, but that's the point — true long runs are aerobic, not anaerobic.

From 4/23 analysis: 37% pace fade across hill reps, HR >175 bpm, perceived exertion 8/10
From 4/25 analysis: 12 km at threshold (high systemic load), zero recovery window

Even though individual session execution is clean (pacing, HR control), the pattern of accumulation is concerning.

**Action required:** 
- **Next 3–4 days (4/26 onward):** Mandatory easy or off days. No structured work, no hard pace.
- **Easy run definition:** Z1–Z2 (110–143 bpm), conversational pace (6:45+ /km), 30–45 min maximum.
- **Or off:** Complete rest day. Walking is fine, no running.

Your body needs the adaptation window. Training stimulus only works if followed by recovery.
