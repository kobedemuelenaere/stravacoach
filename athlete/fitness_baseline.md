## Observations

1. **Fitness is clearly improving.** The pace-to-HR ratio improved markedly across April. Running 1 min/km faster at similar HR in 4 weeks is a real, meaningful gain — not noise.

2. **HR zones are running high.** Most "easy" runs have avg HR in Z3 (143–162 bpm, >75% max HR). This is above what most coaches would call easy. Either the athlete's maxHR is higher than the default 190 assumed, or true easy pace is slower than what's being run, or a mix of both. Worth calibrating once a goal is set.

3. **No HR data for the first three weeks** (March). Five runs are missing HR entirely — the training during this period can only be assessed on pace and perceived exertion, which limits baseline accuracy. RPE was 4–6 throughout, suggesting aerobic intent.

4. **Long run progression is solid.** Three weekly long runs, each stepping up distance. The 4/25 long run (12.1 km, zero HR drift over 76 mins) shows good aerobic base.

5. **Training is varied and includes terrain.** The Karel Sabbeberg hill repeats and dune run add valuable neuromuscular and strength stimulus. The athlete clearly enjoys varied running — this is a strength.

## What the coach should know

- **No HR monitor data in March:** Five out of the first six runs have no HR recorded. This limits the fitness picture for the start of the period. The data gets much richer from April 1 onward.
- **All runs use Mizuno Wave Inspire 2.0** (stability shoe) except the hill/trail session (Salomon xtwings). Suggests the athlete is either overpronating or prefers support — worth flagging if transitioning to faster racing shoes.
- **The "9 slow laps" note on 4/15:** The birthday test run followed an extended slow warm-up, yielding the fastest and most efficient run of the block (5:41/km, eff index 0.036). This warm-up protocol appears to unlock faster running — worth building into tempo sessions.
- **Volume is modest.** Six weeks averaged ~17.8 km/week, last 4 weeks ~19.5 km/week. There is significant headroom to build toward race-training volumes, but any big jumps should be gradual — current fitness is building, not established.
- **No injury mentions** in any Strava descriptions. Clean block.
